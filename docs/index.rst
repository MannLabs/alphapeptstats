Welcome to alphapeptstats's documentation!
======================================

AlphaPeptStats is an open-source package for analyzing mass spectrometry-based proteomics data.

AlphaPeptStats was developed to in order to simplify and standardize the process of analyzig complex datasets. 
Hereby AlphaPeptStats supports proteomics data generated by `AlphaPept`, `DIA-NN`, `FragPipe` and `MaxQuant`. 
The tool allows a structured workflow like importing data, preprocessing data and visualization.
